# Responsive Layout Using Flexbox

## Description
This project demonstrates a responsive webpage layout built using CSS Flexbox. It includes a navigation bar and a card layout that automatically adjusts for different screen sizes.

## Features
- Responsive navigation bar
- Flexible card layout
- Mobile-friendly design
- Clean UI

## Technologies Used
- HTML
- CSS (Flexbox)

## How to Run
1. Clone the repository
2. Open index.html in your browser
